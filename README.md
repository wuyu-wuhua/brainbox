# AI集合站

一个集成多种AI服务的现代化平台，支持AI对话、AI绘画、AI阅读等功能，拥有简洁的左右分栏布局，支持深浅色模式切换与中英文切换，适配PC与移动端，便于后续对接后端API。

## 技术栈
- Next.js
- React
- Chakra UI
- react-i18next
- react-icons

## 启动方式

```bash
npm install
npm run dev
```

## 目录结构

- `/pages`：页面文件
- `/components`：可复用组件
- `/locales`：多语言资源
- `/styles`：全局样式和主题
